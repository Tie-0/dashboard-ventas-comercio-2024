# Dashboard de Ventas — Comercio 2024

Análisis y dashboard interactivo de ventas de un comercio de indumentaria usando datos simulados.

## Herramientas
- Python (pandas, numpy, gspread)
- Google Colab
- Google Sheets
- Google Looker Studio

## Qué incluye
- Generación y limpieza de dataset de 500 ventas
- Análisis por categoría, ciudad y período
- Dashboard interactivo con métricas clave

## Dashboard interactivo
👉 [Ver dashboard en Looker Studio](https://datastudio.google.com/reporting/b48baccf-c293-4f0c-a352-39a65a0d1b56)

## Principales hallazgos
- Ingresos totales: $2,136,000
- Unidades vendidas: 178
- Categoría con mayor ingreso: Calzado
- Las 4 ciudades muestran participación pareja (17-34%)
